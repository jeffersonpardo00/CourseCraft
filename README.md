# CourseCraft
Many people want to help educate children: parents, volunteers, community leaders, migrants, churches, NGOs, tutors without formal training.
But they don’t know:  what to teach, in what order, at what difficulty, or how to adapt to the child’s progress.

*CourseCraft* is the solution.
An AI-assisted curriculum guidance for non-professional educators.
